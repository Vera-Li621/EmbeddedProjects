# EmbeddedProjects
嵌入式作品集合 | ESP32-S3 + 巴法云 MQTT 物联网项目

## ✨ 项目介绍
本项目基于 ESP32-S3 开发板，实现了**温湿度数据采集 + MQTT 协议上报 + 巴法云平台可视化**的完整物联网流程，可用于智能环境监测、远程控制等场景。

## 🛠️ 技术栈
- **硬件**：ESP32-S3 开发板、DHT11/DHT22 温湿度传感器
- **开发环境**：Arduino IDE 2.3.8
- **通信协议**：MQTT
- **云平台**：巴法云 (Bemfa)
- **语言**：C/C++ (Arduino)
- 
## 🚀 功能亮点
- 稳定连接 WiFi 并通过 MQTT 上报温湿度数据
- 接入巴法云平台，支持小程序/网页端实时查看数据
- 代码简洁易扩展，可新增继电器控制、OLED 本地显示等功能
- 已完成完整工程化流程：代码开发 → 调试运行 → GitHub 托管

## 📝 使用说明
1.  硬件接线：将温湿度传感器连接至 ESP32-S3 指定 GPIO 引脚
2.  代码配置：修改 `sketch_mar12a.ino` 中的 WiFi 账号密码、巴法云主题 ID
3.  编译上传：使用 Arduino IDE 编译并烧录至 ESP32-S3
4.  数据查看：登录巴法云平台，查看实时上报的温湿度数据
